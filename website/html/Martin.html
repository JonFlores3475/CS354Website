Example of Limited Duck Typing:

This example reviews the unique implementation of duck typing in TypeScript. Ordinarily, JavaScript would allow any free usage of duck typing with no error checking during compilation-time. However, with TypeScript's gradual typing system, we can swap between strict vs loose typing as it benefits us. This can allow for unique source code as seen below:

Problem Statement:

We need to validate a list of animals and what charcteristics they have. We may have a mixture of attributes we will always mandate and attributes we don't need to mandate.
